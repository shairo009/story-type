# Skill: Premium Hindi Webcomic Automation
Goal: Generate consistent character hand-drawn webcomics in Hindi.

## Rules for Story Engine:
1. **Language**: Narration in Hindi (Devanagari).
2. **Visual Style**: "Hand-drawn webcomic style, minimalist line art, flat color shading, clean ink outlines, expressive cartoon aesthetic, simple background".
3. **Character Consistency (The Secret)**:
   - Always define two main characters:
     - **Mona**: A girl with short black hair, wearing a pink shirt with a purple stripe.
     - **Andy**: A boy with brown curly hair, wearing a white shirt and a red tie, with round glasses.
   - Use these EXACT names and descriptions in every single image prompt.
4. **Composition**: Focus on expressions and simple backgrounds to keep it clean.

## Prompt Template for FLUX:
"[Scene Action] in hand-drawn webcomic style, minimalist line art, flat color shading, clean ink outlines, expressive cartoon aesthetic, simple background, 2D comic art, consistent character design."
